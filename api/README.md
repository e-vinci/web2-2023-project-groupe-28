# BattleShip - Backend

## Description

Backend of our 2023 web project named BattleShip.

### Prerequisites 

-   [Node JS / NPM](https://nodejs.org/en/)

WARNING !! Node version should at least be Node --version 18

### Installing 

Install dependencies

```
npm i 
```

## Getting Started

Run the app

```
npm start
```
